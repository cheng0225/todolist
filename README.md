## Getting Started

1. Navigate to your project directory in the terminal.

2. To run your application in development mode, use the following command:

   ```
   wails3 dev
   ```

   This will start your application and enable hot-reloading for both frontend and backend changes.

3. To build your application for production, use:

   ```
   wails3 build
   ```

   This will create a production-ready executable in the `build` directory.

## Exploring Wails3 Features

Now that your project is set up, you can explore Wails3:

1. **Check out the examples**: Visit the `examples` directory in the `v3/examples` directory in the Wails repo for sample applications.

2. **Run an example**: Navigate to the example's directory and use:

   ```
   go run .
   ```

3. **Documentation**: [Wails3 documentation](https://v3.wails.io/)

4. **Community**: [Wails Discord](https://discord.gg/JDdSxwjhGf) or [Wails discussions on GitHub](https://github.com/wailsapp/wails/discussions)

## Project Structure

- `frontend/`: Frontend code (HTML, CSS, JavaScript/TypeScript)
- `main.go`: Go backend entry point
- `app.go`: Application structure and methods
- `wails.json`: Wails project configuration
